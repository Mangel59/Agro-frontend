Module: GridKardexNuevo

GridKardexNuevo - Componente para la gestión de los items del Kardex. Este componente permite al usuario visualizar, agregar, actualizar y eliminar items de Kardex en una grilla. Proporciona un modal para la entrada y edición de datos, además de validación automática para asignar un `kardexID`.
